# Travel-Guide
基于Spring Boot和Vue.js的旅游出行指南平台，为用户提供全面的旅游信息和个性化的出行建议。用户可以浏览景点、酒店、餐饮和活动，查看评论和评分，制定行程规划。同时，平台支持用户分享旅行体验和旅行路线，增强社交互动。通过简洁直观的界面设计，提升了用户的旅行决策效率，帮助他们更好地探索和享受目的地的魅力。
